# 乗り換え案内アプリ（フロントエンド）

# 技術スタック
Svelte

# 画面
① 検索トップ画面
- 遷移条件
    - アプリを開いたとき
- 構成要素
    - 出発駅入力フォーム
    - 到着駅入力フォーム
    - 出発の日付時刻入力フォーム（デフォルトは現在時刻）
    - 検索ボタン

② 検索結果一覧画面
- 遷移条件
    - ①で検索ボタンが押された時
- 構成要素
    - 検索条件
        - 出発駅→到着駅
        - 出発時間

    - 検索結果カード（5つくらい）
        - 出発時間→到着時間
        - 所用時間
        - 乗り換えの概要（出発駅 - (路線名) ―乗換駅 - (路線名) - 乗換駅 - (路線名) - 到着駅 のようなイメージ）

③ 検索結果詳細画面
- 遷移条件
    - ②のカードを押された時
- 構成要素
    - ②のカードの内容の詳細
        - 出発時間→到着時間
        - 所用時間
        - 乗り換えの概要（出発駅 - 通過駅 - 通過駅 ... ―乗換駅 - .... ）
        -
